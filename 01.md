## 초기 설치시 git 세팅

1. 터미널에 아래 코드를 순서대로 작성
git config --global user.name '이름'
git config --global user.email ''

## github에 코딩파일 업로드
1. 내 프로필에서 repositories 클릭
2. new 클릭
3. repository name에는 코딩파일 제목 작성
4. create repository 클릭
5. vs code 좌측 패널에서 git 아이콘 클릭
6. initialize repository 클릭
7. 업로드할 파일만 + 눌러서 staged changes로 이동
8. message 입력칸에 어떤것이 변경되었는지 입력하고 commit 버튼 클릭
9. 터미널에 아래 코드 순차적으로 붙여넣기
git remote add origin --------------
git remote add origin
git push -u origin main
